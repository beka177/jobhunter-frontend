# JobSearch — заметки для Claude Code

Дипломный проект автора. Подробности живут в долгой памяти, не дублирую здесь.

## Git workflow (важно)

Автор разрешил коммитить и пушить **без подтверждения** в репозитории фронта (этот) и бэка (`c:\OSPanel\domains\jobsearch`).

Правила:
- Коммить после каждой завершённой логической задачи (фича / багфикс / рефакторинг).
- Сообщения коммитов — **на английском**, в нижнем регистре, в стиле автора:
  - `feat: <короткое описание>` — для новых фич
  - `fix bugs` / `bug fixes` — для мелких багфиксов
  - Без тела, без `Co-Authored-By`, без эмодзи.
- Затронули и фронт, и бэк — делай отдельные коммиты в каждом репо.
- Пуш в `origin main` — сразу после коммита.

Исключения, на которые **всё-таки спроси**:
- destructive операции (`git reset --hard`, `push --force`, удаление веток),
- если в diff попадают `.env`, дампы БД, ключи, креды.

**Не коммить**:
- `.claude/settings.local.json` (мои локальные настройки),
- файлы, которые автор не просил трогать (например, `api/seed.php` был удалён автором — это не моя зона).

## Структура проекта

- **Фронт:** React 18 + Vite + Tailwind (через npm, не CDN). State-based routing в [src/App.jsx](src/App.jsx).
- **Бэк:** vanilla PHP + PDO в `c:\OSPanel\domains\jobsearch\api\`, доступ через `http://jobsearch/api`.
- **БД:** MySQL `jobsearch` в OSPanel, доступ через phpMyAdmin.

SQL-миграции для новых таблиц складываем в `sql/` фронта, автор импортирует через phpMyAdmin вручную.

## Стиль кода

- Комментарии в JSX и PHP — по-русски (автор пишет на русском).
- Идентификаторы и коммиты — на английском.
- Tailwind-классы инлайнятся в JSX, отдельных CSS-модулей нет.
- Тосты вместо `alert()` — используй `useToast()` из [src/toast.jsx](src/toast.jsx).
- Дебаунс поиска через `useDebounce` из [src/hooks.js](src/hooks.js).
